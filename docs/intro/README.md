# AutoSnap 参赛宣传材料

![AutoSnap 智能截图归档助手封面](./autosnap-cover.png)

## 创意主题

AutoSnap：让截图自动归档、随时找回

## 作品简介

AutoSnap 是一款面向个人效率提升的 AI 截图归档助手。用户无需改变原有截图习惯，继续使用系统自带截图方式即可；AutoSnap 会在后台自动捕获截图，将图片按时间规范归档，并建立本地索引，方便后续按时间、关键词、内容类别快速回溯。

在基础模式下，AutoSnap 不依赖云端服务或 API Key，也能完成截图保存、去重、时间线浏览和本地搜索。开启多模态 AI 后，系统可以进一步理解截图内容，自动生成类别、标题、标签与摘要，让大量零散截图变成可检索、可整理、可长期保存的个人资料库。

它适合日常办公、学习记录、代码调试、网页资料收藏、聊天信息回溯等场景，目标是解决“截图很多、命名混乱、事后找不到”的普遍痛点。

## 项目说明

AutoSnap 的核心理念是“不打扰用户原有截图流程”。它不是新的截图工具，而是截图完成后的智能归档层。

主要能力：

- 自动监听截图目录和剪贴板截图
- 按日期生成规范化归档目录
- 自动去重，避免重复保存
- 提供时间线视图快速回看历史截图
- 支持关键词搜索截图记录
- 可选 AI 内容理解，自动生成分类、标签和摘要
- 无 AI 模式下仍可作为本地截图管理工具使用

## 一句话亮点

把每天散落的截图，变成一个可搜索、可回溯、可长期沉淀的个人视觉档案库。
